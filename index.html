No começo desse mês, iniciamos nossos primeiros passos como desenvolvedores na Trybe. Até esse presente projeto, tivemos contato com toda a estrutura fundamental do sistema operacional Linux, utilizando seus principais comandos no terminal.
Além disso, utilizamos diariamente Git e GitHub para o versionamento de todos nossos projetos e exercícios, que nesse começo de curso foram focados em HTML e CSS para estruturar e estilizar nossos sites.

"A princípio, enxergava a Trybe como um curso que disponibilizaria apenas o conteúdo técnico proposto no currículo. Felizmente, eu estava errado.
Não apenas as hard skills, também desenvolvemos nossa carreira profissional com soft skills e a construção de uma nova família"
