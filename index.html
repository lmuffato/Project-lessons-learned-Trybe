O inicio da tragetória

Nesse mês de fevereiro, inicio-se a minha trajetória para a formação em Desenvolvimento de Sistemas na Trybe. Em duas semanas, já pude aprender um elevado número de habilidades. Na primeira semana de aula foi aprendido como operar no ambiente linux, os principais comandos do terminal. A partir da segunda até a atual, foi introduzido conceitos do Git e GitHub que são importantes ferramentas para versionamento de código e programação paralela entre membros de equipes. Por fim foram introduzidas as linguagens HTML e CSS que são as bases de qualquer página web.
