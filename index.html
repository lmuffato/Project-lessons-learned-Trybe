Meu aprendizado

Essa é minha terceira semana de estudo na Trybe e estou aprendendo muito sobre diversos temas, começando pelo sistema Linux, o qual nunca havia utilizado antes e estou adorando, principalmente por sua inicialização, que é muito mais rápido que o do Windows. Depois disso, comecei a aprender sobre como utilizar o terminal so Linux, o que é essencial para realizar as atividades. E para essas atividades, também aprendi a usar o Github, para controlar o versionamento dos meus trabalhos e nunca mais precisar salvar várias versões semi finalizadas.

Uma passagem sobre meu aprendizado

Graças ao meu aprendizado sobre Linux, eu consegui "salvar" um computador antigo que estava tendo alguns problemas de desempenho. Como aprendi que um Linux é um sistema "leve", ótimo para rodar nesses casos, decidi usa-lo nesse computador e estou tendo um ótimo resultado.
