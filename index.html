Meu aprendizado

Desafios nos fazem crescer! Esse foi um dos motivos de eu iniciar minha trajetória na Trybe. O aprendizado aqui se inicia desde o básico, aquilo que é primordial para um bom desenvolvimento no decorrer do curso. Somente após ver e compreender o fundamental sobre Linux, Unix, Bash, Internet, Imputs e Outputs, Controle de arquivos, Internet, Git e outros, pudemos finalmente avançar para estruturas importantes como HTML, CSS e, muito em breve, JavaScript - itens contemplados no módulo I do curso.
